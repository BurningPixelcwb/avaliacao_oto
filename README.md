# avaliacao_oto
Scripts para avaliação OTO
--------- Criar um ambiente virtual ---------
python -m venv venv_oto

--------- Levantando ambiente virtual ---------
venv_oto\Scripts\activate

--------- Instalando dependencias ---------
pip install -r avaliacao_oto\requirements.txt

--------- Rodando ---------
cd avaliacao_oto
python index.py
